# Dataset Description

The dataset contains simulated real-world driving telemetry data.

## Key Columns
| Column | Description |
|--------|-------------|
| speed | Vehicle speed (km/h) |
| acceleration | Acceleration rate (m/s²) |
| steering_angle | Steering wheel angle (degrees) |
| trip_duration | Time duration of trip (seconds) |
| trip_distance | Trip distance (km) |
| brake_usage | Brake intensity |
| lane_deviation | Lateral lane deviation |
| weather_conditions | Weather during driving |
| anomalous_event | Risk label (0/1) |

## Dataset Size

- Total samples: 120,000 rows
- Binary classification task

## Source
Simulated driving behaviour dataset provided for MSc project use.
